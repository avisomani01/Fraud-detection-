# Fraud-detection-
Fraud detection system using a hybrid machine learning approach combining XGBoost, Isolation Forest, and ensemble models. Includes feature engineering, cost-sensitive optimization, and a Streamlit dashboard for visualization, evaluation, and real-time prediction on imbalanced financial transaction data.

## Features
- XGBoost Model
- Isolation Forest
- Ensemble Learning

## Tech Stack
- Python
- Scikit-learn
- Pandas

## How to Run
pip install -r requirements.txt
python app.py
