# 💳 Fraud Detection System

## 📌 Overview

This project is a machine learning-based fraud detection system designed to identify fraudulent financial transactions using advanced models and ensemble techniques.

## 🚀 Features

* XGBoost-based classification
* Isolation Forest for anomaly detection
* Ensemble learning for improved accuracy
* Data preprocessing and feature engineering
* Model evaluation with performance metrics

## 🛠 Tech Stack

* Python
* Scikit-learn
* XGBoost
* Pandas, NumPy
* Flask (if used)

## 📂 Project Structure

backend/

* app.py
* model_training.py
* evaluation.py
* data_ingestion.py

## ▶️ How to Run
